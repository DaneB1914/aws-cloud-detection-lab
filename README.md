# AWS Cloud Attack & Detection Engineering Lab

This project documents the creation of a beginner-friendly AWS cloud security lab focused on detection engineering, cloud misconfiguration analysis, and safe attack simulation.

## Project Goals

- Learn AWS fundamentals
- Configure CloudTrail, GuardDuty, and VPC Flow Logs
- Simulate suspicious AWS activity
- Write detections for cloud attack behavior
- Document findings and remediation steps

## Skills Demonstrated

- AWS IAM
- AWS S3
- AWS CloudTrail
- AWS GuardDuty
- AWS VPC Flow Logs
- Cloud detection engineering
- Python log analysis
- Vulnerability analysis
- Security reporting
